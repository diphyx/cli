## diphyx dashboard

Open the DiPhyx platform dashboard

### Synopsis

Open the DiPhyx platform dashboard with single sign-on

```
diphyx dashboard [flags]
```

### Options

```
  -N, --no-open   skip opening the dashboard in the default web browser
  -h, --help      help for dashboard
```
